SQL

SQL es el lenguaje para administrar y consultar información de bases de datos relacionales.

SQL “Structured Query Language”

DDL (Data Definition Language)

El lenguaje de definición de datos (DDL) es un subconjunto de SQL. Es un lenguaje para describir los datos y sus relaciones en una base de datos.

Tipo de Datos:
Char

Clona un repositorio Git existente desde una URL.

Varchar

Agrega archivos al área de preparación (staging) para ser incluidos en el próximo commit.

Int

Crea un commit con los archivos en el área de preparación y un mensaje descriptivo.

Date

Muestra el estado actual del repositorio, incluyendo archivos modificados, agregados y pendientes de confirmar.

Text

Sube los commits locales al repositorio remoto.

Modificadores:
auto_increment

El valor se va incrementando automáticamente en cada registro. Aplica para números enteros.

Default

Coloca un valor por defecto, el valor se coloca justo detrás de esta palabra. Aplica para todos tipo de campo excepto TEXT y BLOB.

not null

Impide que un campo sea nulo.

Primary Key

Hace que el campo se considere llave primaria.

Unique

Evita la repetición de valores.

Modificación de objetos creados:
Modify

Modifica un campo, esto puede ser el tipo de dato y/o ancho del campo.

Change

Modifica el nombre de un campo.

Rename

Cambia el nombre a una table.

Drop

Borra Columnas.

Add

Adiciona columnas a una tabla, es importante informar la ubicación de la nueva columna utilizando AFTER.
